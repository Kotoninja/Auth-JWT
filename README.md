
## Description
A simple project for user authentication via JWT using React + DRF
## Getting Started

### Dependencies

* Python 3.12 +
* Docker
* Git

### Installing

1. Clone the repository
```
git clone git@github.com:Kotoninja/Auth-JWT.git
cd Auth-JWT
```
2. Env configuration

```
touch .env
```
Open the .env file and paste the entire contents of the .env.example file (if you want to start the project quickly run ```cp .env.example .env```, otherwise see the .env.dist file and set up your own .env file).

### Executing program

* Run ```docker compose run --build```
* Open your browser and go to http://localhost:3000
